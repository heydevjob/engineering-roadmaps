`13 projects` · `Debug · Build · Tune · Optimize · Design` · `Junior → Senior`

Real LLM features to build and fix - integrations, prompts, RAG, agents, and evals. Each one proves a specific AI-engineering skill and produces something you can put on a portfolio.
