# Fullstack Engineer Projects

`13 projects` · `Debug · Build · Optimize · Harden · Design` · `Junior → Senior`

Real features to build and fix across the whole stack - React, the API, the database, and the seams between them. Each one proves a specific fullstack skill and produces something you can put on a portfolio. Fullstack is about owning a feature end to end, so these run from frontend state bugs to real-time backends. Every project is grounded in a real [HeyDevJob](https://heydevjob.com/fullstack) ticket: the actual code, the real symptom, and a live cloud workspace where you build or fix it.

> [!NOTE]
> **Project types**
> 🔨 **Build** - create it from nothing · 🔍 **Debug** - find and fix what's wrong · ⚡ **Optimize** - make it fast and smooth · 🛡️ **Harden** - make it secure and reliable · 📐 **Design** - architect a feature

Pairs with the [Fullstack Engineer Roadmap](../../roadmaps/fullstack.md): the roadmap tells you what to learn, these are where you prove it.
